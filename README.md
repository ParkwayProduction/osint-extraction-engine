# osint-extraction-engine
Automated web-scraping engine utilized to harvest SpaceX launch telemetry from distributed wiki nodes via BeautifulSoup

[ MISSION_OVERVIEW ]
Deployment of a specialized web-scraping framework designed to extract historical Falcon 9 telemetry from Wikipedia. This module serves as the primary data-ingest layer for secondary predictive and geospatial intelligence pipelines.

[ EXTRACTION_PROTOCOLS ]
BeautifulSoup Ingest: Deep-parsing of collapsible HTML plain-row headers.

Data Normalization: Cleansing of inconsistent UTF-8 strings and reference noises.

DataFrame Export: Automated conversion to structured .CSV files for downstream analytical missions.
