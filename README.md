# [ MODULE : OSINT_EXTRACTION_ENGINE ]

### [ MISSION_OVERVIEW ]
Automated web-scraping framework designed to extract historical mission telemetry from distributed wiki nodes. This engine serves as the primary OSINT layer for harvesting raw data into structured datasets for downstream analytical missions.

### [ EXTRACTION_PROTOCOLS ]
* **BeautifulSoup Ingest:** Deep-parsing of collapsible HTML plain-row headers to target specific telemetry nodes like booster landing and payload mass.
* **Data Normalization:** Automated cleansing of inconsistent UTF-8 strings and reference noise found in raw web sources.
* **Serialization:** Seamless conversion of harvested OSINT into structured Pandas DataFrames for immediate ingest.
